🤖 Installation AI CLI 🐧 Commandes Linux

🐧 COMMANDES TERMINAL LINUX

Référence des principales commandes Linux organisées par catégorie. Cliquez sur l'icône de copie pour copier une commande.

📁 Navigation & Fichiers

Lister les fichiers

Affiche le contenu du répertoire courant avec détails et fichiers cachés.

ls -la

Changer de répertoire

Se déplace dans le répertoire spécifié.

cd /chemin/vers/dossier

Répertoire courant

Affiche le chemin absolu du répertoire actuel.

pwd

Créer un dossier

Crée un ou plusieurs répertoires (avec parents si nécessaire).

mkdir -p dossier/sous-dossier

Supprimer fichier/dossier

Supprime récursivement un dossier et son contenu.

⚠ Prudence rm -rf dossier/

Copier

Copie un fichier ou un dossier vers une destination.

cp -r source/ destination/

Déplacer / Renommer

Déplace ou renomme un fichier ou dossier.

mv ancien_nom nouveau_nom

Chercher un fichier

Recherche un fichier par son nom dans un répertoire.

find / -name "fichier.txt"

Arborescence

Affiche l'arborescence des répertoires en mode graphique.

tree -L 2

📝 Affichage & Édition

Afficher un fichier

Affiche le contenu complet d'un fichier.

cat fichier.txt

Parcourir page par page

Permet de faire défiler un fichier long page par page.

less fichier.txt

Premières lignes

Affiche les 20 premières lignes d'un fichier.

head -n 20 fichier.txt

Dernières lignes

Affiche les 20 dernières lignes (ou en temps réel avec -f).

tail -n 20 fichier.txt

Éditer avec nano

Ouvre un fichier dans l'éditeur nano (simple d'utilisation).

nano fichier.txt

Éditer avec vim

Ouvre un fichier dans vim (puissant, courbe d'apprentissage élevée).

vim fichier.txt

Rechercher dans un fichier

Cherche un motif dans un fichier (insensible à la casse).

grep -i "motif" fichier.txt

Rechercher récursivement

Cherche un motif dans tous les fichiers d'un répertoire.

grep -r "motif" ./dossier/

⚙️ Processus & Système

Processus en cours

Affiche tous les processus avec utilisation CPU et RAM.

ps aux

Moniteur interactif

Interface interactive pour surveiller les processus en temps réel.

htop

Tuer un processus

Termine un processus par son identifiant (PID).

kill -9 <PID>

Tuer par nom

Termine tous les processus portant ce nom.

pkill nom_processus

Utilisation disque

Affiche l'espace disque utilisé et disponible.

df -h

Taille d'un dossier

Calcule la taille d'un répertoire de façon lisible.

du -sh dossier/

Mémoire disponible

Affiche l'utilisation de la RAM et du swap.

free -h

Infos système

Affiche la version du noyau Linux et l'architecture.

uname -a

Uptime

Temps de fonctionnement du système et charge moyenne.

uptime

🌐 Réseau

Interfaces réseau

Affiche les interfaces réseau et leurs adresses IP.

ip a

Ping

Teste la connectivité vers un hôte (5 paquets).

ping -c 5 google.com

Connexions actives

Liste les ports ouverts et connexions réseau actives.

ss -tuln

Télécharger un fichier

Télécharge un fichier depuis une URL.

wget https://exemple.com/fichier

Requête HTTP

Effectue une requête HTTP et affiche la réponse.

curl -I https://exemple.com

Tracer la route

Affiche le chemin réseau vers un hôte.

traceroute google.com

🔐 Permissions & Utilisateurs

Changer les permissions

Définit les droits en lecture/écriture/exécution.

chmod 755 fichier

Changer le propriétaire

Change l'utilisateur et le groupe propriétaire d'un fichier.

chown user:groupe fichier

Utilisateur courant

Affiche le nom de l'utilisateur connecté.

whoami

Changer d'utilisateur

Bascule vers un autre compte utilisateur.

su - nom_utilisateur

Commande admin

Exécute une commande avec les droits super-utilisateur.

⚠ Prudence sudo commande

Créer un utilisateur

Crée un nouvel utilisateur sur le système.

sudo useradd -m nom_utilisateur

📦 Compression & Archives

Créer une archive tar.gz

Compresse un dossier en archive .tar.gz.

tar -czf archive.tar.gz dossier/

Extraire tar.gz

Décompresse une archive .tar.gz dans le répertoire courant.

tar -xzf archive.tar.gz

Compresser en zip

Crée une archive zip d'un dossier.

zip -r archive.zip dossier/

Décompresser un zip

Extrait le contenu d'une archive zip.

unzip archive.zip

📥 Gestion des Paquets (apt)

Mettre à jour les listes

Rafraîchit la liste des paquets disponibles.

sudo apt update

Mettre à jour les paquets

Met à jour tous les paquets installés.

sudo apt upgrade -y

Installer un paquet

Installe un logiciel depuis les dépôts officiels.

sudo apt install nom_paquet

Désinstaller un paquet

Supprime un paquet et ses fichiers de configuration.

sudo apt remove --purge nom_paquet

Chercher un paquet

Recherche un paquet dans les dépôts.

apt search nom_paquet

Nettoyer le cache

Supprime les fichiers téléchargés inutiles.

sudo apt autoremove && sudo apt clean

🛠️ Divers & Astuces

Historique des commandes

Affiche les dernières commandes saisies.

history

Effacer le terminal

Nettoie l'affichage du terminal.

clear

Manuel d'une commande

Affiche la documentation complète d'une commande.

man ls

Alias rapide

Crée un raccourci pour une commande longue.

alias ll='ls -la'

Variable d'environnement

Affiche toutes les variables d'environnement.

env

Date et heure

Affiche la date et l'heure actuelles du système.

date

Redirection de sortie

Redirige la sortie d'une commande vers un fichier.

ls -la > sortie.txt

Pipe (enchaînement)

Passe la sortie d'une commande en entrée d'une autre.

ls -la | grep ".txt"
