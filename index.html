Installer des outils IA en CLI

Voici des exemples rapides pour installer des outils d'IA en ligne de commande avec npm. Assure-toi d'avoir Node.js et npm déjà installés.

Google Gemini CLI

Installe l'outil CLI Gemini globalement :

npm install -g @google/gemini-cli

OpenAI Codex CLI

Installe Codex en global :

npm install -g @openai/codex

Astuce : si la commande globale échoue, essaie avec droits admin (sudo sur Linux/macOS) ou exécute ton terminal en administrateur sur Windows.
